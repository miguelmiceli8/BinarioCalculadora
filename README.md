# BinarioCalculadora
Projeto Desenvolvido em parceria com Antonio Modenese
Durante o 2º período da graduação, na disciplina de Circuitos Lógicos e Digitais, tive contato com conceitos fundamentais de sistemas de numeração e conversão entre bases, com foco em conversões de números binários para decimal e hexadecimal.

A partir desse conteúdo, eu e um colega de faculdade desenvolvemos um site voltado à conversão de números binários, permitindo a transformação desses valores para as bases decimal e hexadecimal. O projeto foi desenvolvido com Python no backend, responsável pela lógica de conversão, utilizando o framework Flask para a integração com o frontend.

No frontend, utilizamos HTML e CSS, com o apoio do framework Tailwind CSS, garantindo uma interface moderna, responsiva e de fácil usabilidade.

O objetivo do projeto foi aplicar, na prática, os conceitos teóricos abordados em sala de aula, além de fortalecer conhecimentos em desenvolvimento web e na integração entre frontend e backend.
